## 🏋🏽 FitLife App

FitLife App es un programa  desarrollado en C que centraliza el control de la salud física. Permite calcular el índice de masa corporal (IMC), genera planes nutricionales y rutinas de ejercicio adaptadas al nivel del usuario, e incluye un cronómetro interactivo para medir el tiempo de entrenamiento, calcular las calorías quemadas y registrar la constancia diaria.
